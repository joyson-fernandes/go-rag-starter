go-rag-starter

A self-hosted, self-documenting RAG help bot template. You're looking at the built-in demo page. The bot running behind the widget (bottom-right bubble) was indexed from the markdown files in this repo's docs/ directory — so it already knows how to answer questions about itself.

Click the bubble in the bottom-right and try:
  • "How do I replace the corpus?"
  • "How do I swap Ollama for OpenAI?"
  • "What is pgvector?"
  • "How do I embed this widget on my own site?"

Embed on your own site

One <script> tag, no build step:

<script src="http://YOUR-HOST:8080/widget.js"
        data-api="http://YOUR-HOST:8080"
        data-title="Ask Acme"
        data-subtitle="Answers from the Acme docs"></script>

Replace the corpus

Drop your own markdown files into docs/, rebuild the Docker image, restart. The service hashes the corpus on startup and re-embeds only when it changed. See docs/01-replace-the-corpus.md for the format.
